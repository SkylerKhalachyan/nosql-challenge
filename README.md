# nosql-challenge
#

### Overview of Project
An evaluation of ratings for establishments across the United Kingdom for a magazine, Eat, Safe, Love. This evaluation will assist the journalists and food critics decide where to focus future articles. 
### Built Utilizing
* PyMongo 
* MongoDB
* Python
* Pandas
* Matplotlib

### User Set Up 
#

### Prerequisites
